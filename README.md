# Data Jobs Dashboard

## Project Overview

This project analyses junior data job postings to understand what skills and tools employers commonly request.

The goal is to create a simple, visual dashboard that turns job posting data into useful insights for someone applying to junior data analyst, data engineer and data science roles.

## Project Goals

- Analyse junior data job postings
- Identify most requested technical skills
- Compare data analyst, data engineer and data science roles
- Create clear charts and dashboard visuals
- Practise communicating insights from data

## Tools

- Python
- pandas
- matplotlib, seaborn or Plotly
- Streamlit, Power BI or Tableau
- GitHub

## Possible Data Columns

- Job title
- Company
- Location
- Remote/hybrid/on-site
- Salary range
- Required skills
- Role category
- Posting date

## Planned Charts

- Most requested skills
- Role type breakdown
- Remote vs hybrid vs on-site jobs
- Skills by role type
- Salary range distribution, if available
- Number of postings by location

## Example Questions

- Which skills appear most often in junior data roles?
- Is SQL or Python more commonly requested?
- How often do AWS, dbt and Power BI appear?
- Which roles are more likely to be remote or hybrid?

## What This Project Demonstrates

- Data cleaning
- Exploratory data analysis
- Visualisation
- Dashboard thinking
- Clear communication of insights
